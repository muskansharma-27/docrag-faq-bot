
# Docrag FAQ Bot

An AI-powered FAQ chatbot application with RAG (Retrieval-Augmented Generation) integration using n8n and connected with a React frontend.

## Features

- AI-powered chatbot interface
- RAG-based document question answering
- n8n workflow integration
- Responsive and interactive frontend
- Real-time chatbot responses
- FAQ and document handling support

## Tech Stack

- React
- TypeScript
- Vite
- n8n
- AI/LLM Integration
- CSS

## Workflow

Frontend → n8n Webhook → AI/RAG Processing → Response to Frontend
